# Hay Suficiente

Wireframe de alta resolución para mostrar la operacividad de una aplicación interactiva planificada para concienciar sobre el hambre en el mundo.

A partir de la afirmación de que EL HAMBRE ES UNA REALIDAD EN UN MUNDO DONDE HAY SUFICIENTE COMIDA PARA TODA LA POBLACIÓN Y PARA ALIMENTAR A 3000 MILLONES MÁS.

## Objetivos

El objetivo de HAY SUFICIENTE es ofrecer una propuesta de solución digital que sirva para concienciar a un sector de la población que de otro modo no accedería a la información relativa al hambre en el mundo y los agentes que la ocasionan.

```bash
El uso de la aplicación es parte de una solución híbrida planteada para conseguir el mayor número de personas concienciadas en la problemática del hambre en el mundo 
```
